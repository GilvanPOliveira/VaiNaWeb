# Aprendendo React

Aplicação web desenvolvida para introdução ao uso de JavaScript e React.

##

## Sobre

Este projeto foi desenvolvido como desafio prático com foco na introdução ao desenvolvimento com React.

A proposta foi aplicar conhecimentos de HTML, CSS e JavaScript em um novo contexto utilizando componentes e estrutura de aplicação moderna.

##

## Objetivo

Consolidar conhecimentos iniciais em React, incluindo:

* criação de aplicações com React
* estruturação de componentes
* organização de projetos frontend
* integração com JavaScript
* uso de boas práticas no desenvolvimento

##

## Funcionalidades

### Interface

* página estruturada com componentes React
* organização visual de conteúdo

##

### Componentização

* separação de responsabilidades em componentes
* reutilização de estruturas

##

### Responsividade

* adaptação para diferentes tamanhos de tela
