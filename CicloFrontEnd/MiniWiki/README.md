# Mini Wiki

Aplicação web desenvolvida para apresentação pessoal utilizando HTML.

##

## Sobre

Este projeto foi desenvolvido como desafio inicial com foco na criação de uma página pessoal.

A proposta foi aplicar conceitos básicos de estruturação de páginas web utilizando HTML.

##

## Objetivo

Consolidar conhecimentos iniciais em desenvolvimento frontend, incluindo:

* estruturação de páginas com HTML
* uso de tags semânticas
* organização de conteúdo
* criação de listas e links

##

## Funcionalidades

### Interface

* página com informações pessoais
* organização de conteúdo em seções

##

### Estrutura

* utilização de header, main e footer
* uso de listas ordenadas e não ordenadas
