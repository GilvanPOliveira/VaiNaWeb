# Vai na Web

Repositório com projetos desenvolvidos durante o curso do instituto Vai na Web.

##

## Sobre

Este repositório reúne os projetos desenvolvidos ao longo do programa, organizados por módulos e níveis de complexidade.

Cada projeto possui seu próprio README, incluindo instruções e, quando disponível, link para deploy.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento web, incluindo:

* desenvolvimento frontend
* desenvolvimento full stack
* consumo e criação de APIs
* organização de projetos
* aplicação prática de tecnologias modernas
