# Ciclo FullStack

Módulo de estudos com projetos desenvolvidos durante o ciclo FullStack.

##

## Sobre

Este diretório reúne os projetos desenvolvidos ao longo do ciclo FullStack, organizados por desafios.

A estrutura demonstra a evolução do aprendizado por meio da resolução prática de problemas e construção de aplicações completas.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento full stack, incluindo:

* desenvolvimento frontend e backend
* consumo e criação de APIs
* organização de projetos
* aplicação prática de tecnologias modernas
