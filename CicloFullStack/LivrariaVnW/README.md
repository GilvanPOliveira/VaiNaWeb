# Livraria Vai na Web

Aplicação web desenvolvida com React para simular uma plataforma de livraria.

##

## Sobre

Este projeto foi desenvolvido como desafio final com foco na construção de uma aplicação completa utilizando React.

A proposta foi aplicar todos os conhecimentos adquiridos ao longo do curso em um projeto estruturado e alinhado a um design definido.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* desenvolvimento de interfaces com React
* estilização com Sass modularizado
* organização de componentes
* aplicação de boas práticas de estruturação
* fidelidade a layout proposto em design

##

## Funcionalidades

### Interface

* layout completo de uma livraria digital
* organização visual de conteúdos e seções

##

### Componentização

* separação de responsabilidades em componentes
* reutilização de estruturas

##

### Responsividade

* adaptação para diferentes tamanhos de tela
