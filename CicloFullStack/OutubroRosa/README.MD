# Outubro Rosa

Aplicação web desenvolvida para conscientização sobre o Outubro Rosa.

##

## Sobre

Este projeto foi desenvolvido como desafio prático com foco na criação de uma interface temática voltada à conscientização sobre a saúde da mulher.

A proposta foi aplicar conceitos de desenvolvimento frontend em um contexto informativo e visual.

##

## Objetivo

Consolidar conhecimentos em desenvolvimento frontend, incluindo:

* estruturação de páginas com HTML
* estilização com CSS e Sass
* construção de layouts responsivos
* organização de código
* aplicação de conceitos visuais em temas informativos

##

## Funcionalidades

### Interface

* página temática de conscientização
* apresentação de conteúdos informativos

##

### Layout

* organização visual focada em clareza e acessibilidade

##

### Responsividade

* adaptação para diferentes tamanhos de tela
